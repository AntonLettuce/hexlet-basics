Подробнее о настройке/ведении контекстной рекламы

  1. Подбор минус-слов. Подбираем слова или словочетания, по которым не должна показываться реклама.
  2. Создание быстрых ссылок, уточнений. Чем больше заполнено объявление, тем выше вероятность клика по нему (CTR).
  3. Установка UTM меток. UTM-метки нужны для отслеживания источника трафика, например: с какой именно ключевой фразы и какого именно места в рекламных местах был произведен переход.
  4. Количество объявлений. Количество уникальных объявлений.
  5. Количество запросов. Количество ключевых запросов, по которым будет запущена реклама.
  6. Настройка систем аналитики. Установка Яндекс метрики/Google Analytics для отслеживания рекламного трафика.
  7. Настройка целей на сайте. Настройка отслеживания целевых действий пользователя, для получения первоначальных данных о конверсии.
  8. Ежедневный мониторинг динамики и эффективности. Проверка работоспособности рекламной кампании.
  9. Маркетинговый аудит. Анализ конкурентов, формулировка торгового и рекламного предложений и т. д.
  10. Тестирование различных текстов объявлений. Написание различных текстов объявлений для получения более полной статистики.
  11. Отдельные кампании для поиска и контекстно-медийной сети. Сам Яндекс рекомендует различные кампании для поиска и РСЯ, необходимый атрибут для хорошей рекламной кампании.
  12. Анализ юзабилити, составление рекомендаций по улучшению конверсии. Анализ удобства сайта может увеличить вашу конверсию в разы, бывает, что один элемент в неправильном месте сайта может сильно занижать ваши продажи.
  13. Настройка ремаркетинга/ретаргетинга. Специальные объявления для пользователей, которые уже посещали ваш сайт.
